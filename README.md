# Not All Neighbours

A first-person psychological horror detective game with fixed-camera perspective.

## Project Setup

### Requirements

- Unity 2022 LTS or later
- Git with LFS support
- 10GB free disk space

### First Time Setup

1. Clone repository with LFS: `git clone --recurse-submodules [repository-url]`
2. Open in Unity Hub
3. Let Unity import all assets
4. Open scene: Assets/Scenes/Development/TestScene
